 분석(예측) 결과로부터 인사이트를 발굴하고 의사결정 방법을 검토 선택하며 데이터 특성에 적합한 차트와 시각화 도구를 선택하는 방법을 계획하는 시나리오를 작성

기능구조 : 분석조직이 없고 업무부서에서 분석을 수행
분산구조 : 조직의 인력을 협업부서에 배치, 빠름
집중형 : 조직내의 별도의 독립적인 분석 조직 구성

분석마스터 플랜 
  - 분석과제 정의
    - 데이터, 분석 방법, 난이도, 분석수행주기, 상세분석 등

  - 분석 과제의 적용밤위 및 방식의 고려요소 ( 분석 적용 범위)
 	    내재화, 분석 데이터 적용 수준, 기술 적용 수준
 - 분석 과제 우선순위 고려요소 ( 분석 수행때 ) - 용전 분R 수
    	실행용이성, 전략적 중요성, 분석 ROI
 -  시급성 판단기준
     전략적 중요도 / 시급성 기준으로 할때 가장 쉬운거 -> 시급성 어려운거 -> 난이도 어려운거
 - 분석 수준 진단 결과 항목
     인력, 분석 업무, 기법
 - 의사나무 만족하는 거 찾기
    - 연관성 높은 거에 영향 많이 받음
    해석, 구현에 쉬움

- 소프트맥스
  인공신경망, 출력합:1
- 시그모이드
  인공신경망, 0~1사이 출력, 독립
신경망모형 / 활성화함수
 - 입력신호를 출력신호로 변환
